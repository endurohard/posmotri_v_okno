               # Проект "Посмотри в окно"

## Описание проекта

Интерактивное веб-приложение для просмотра видео из разных городов и времени суток. Приложение позволяет искать видео по названию города и выбранному времени суток (утро, день, ночь).

## Функциональность

- Просмотр видео в основном проигрывателе
- Поиск видео по городу и времени суток
- Список карточек с превью видео
- Переключение между видео по клику на карточку
- Подгрузка дополнительных видео через кнопку "Показать ещё"
- Прелоадеры во время загрузки контента
- Обработка ошибок с выводом сообщений

## Технологии

- HTML5
- CSS3 (Flexbox)
- JavaScript
- Кастомные веб-шрифты (Oswald, Fira Sans Condensed)

## Структура проекта

```
posmotri_v_okno/
├── fonts/              # Шрифты проекта
├── scripts/            # JavaScript файлы
├── styles/             # CSS файлы
│   ├── style.css      # Основные стили
│   ├── preloader.css  # Стили прелоадера
│   └── error.css      # Стили ошибок
└── index.html         # Главная страница
```

## Особенности реализации

- Использование логических свойств CSS (inline-size, block-size)
- Кастомные чекбоксы через псевдоэлементы
- Кастомный скроллбар для списка видео
- Адаптивные состояния интерактивных элементов (:hover, :active, :focus-visible)
- Доступность (visually-hidden для скрытого контента)
